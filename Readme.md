Adobe Illustrator Calendar Premaker Script

It features a modern interface and flexible layout options to help you work faster and more efficiently.

How to Use

Open Adobe Illustrator

Go to File > Scripts > Other Script...

Select Calendar Premaker.jsx

In the settings dialog:

Year: Enter the calendar year (e.g. 2026)

General: Choose language and week start day

Layout: Set horizontal and vertical gaps

Customization: Edit month/day names and add holidays
(Holiday format: D/M, separated by commas)

Click OK

The calendar will be generated as fully editable vector elements in a new document.
